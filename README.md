# index-tianbo-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容互不关联，各自独立，旨在集中存储一些静态页面资源。

## 项目简介

这是一个轻量级的页面归档仓库，主要存放一些零散的 HTML 文件。每个页面都有其独立的用途或主题，不依赖外部数据库或后端服务，可直接通过浏览器打开访问。

## 目录说明

- `./` — 仓库根目录下直接存放各 HTML 页面文件。
- 页面文件名通常能反映其大致内容或功能。
- 如有子目录，则用于存放对应页面的附属资源（如 CSS、JS、图片等）。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接通过 GitHub Pages 或其他静态托管服务访问。
- 页面之间无链接关系，浏览时请直接打开对应文件。
- 部分页面可能包含简单的交互或样式，但整体以内容展示为主。

## 维护说明

- 本仓库不定期更新，新增或修改页面时，会直接提交至主分支。
- 欢迎通过 Issue 提出建议或报告问题，但请注意本仓库仅为个人归档用途。
- 不保证所有页面长期可用或内容持续更新。

## 使用方式

1. 克隆本仓库到本地。
2. 直接在浏览器中打开所需的 HTML 文件。
3. 或通过 GitHub Pages 功能在线访问（需自行启用）。

## 许可

本仓库内容仅供个人参考与归档，不涉及商业用途。如需引用或分发，请注明出处。
